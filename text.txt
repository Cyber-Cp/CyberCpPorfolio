# 1. Create a new repository on GitHub
# - Go to https://github.com/
# - Click "New" to create a new repository
# - Enter repository name (e.g., my-website)
# - Choose visibility (Public/Private)
# - Do NOT initialize with README
# - Click "Create repository"

# 2. Clone the repository to your local machine
git clone https://github.com/your-username/my-website.git
cd my-website

# 3. Add your website files to this folder
# (Place index.html, CSS, JS files here)

# 4. Stage and commit your files
git add .
git commit -m "Initial commit with website files"

# 5. Push to GitHub
git push origin main
# (Use 'master' if your branch is named master)

# 6. Deploy on Netlify
# - Go to https://www.netlify.com/
# - Sign in or create an account
# - Click "New site from Git"
# - Choose "GitHub" as your provider
# - Authorize Netlify to access your repositories
# - Select your repository (my-website)
# - Configure build settings:
#    - Branch to deploy: main (or master)
#    - Build command: leave empty if static
#    - Publish directory: (e.g., ./ or build/ if applicable)
# - Click "Deploy site"

# 7. Update your website
# - Make changes locally to your files
git add .
git commit -m "Update website"
git push origin main
# Netlify will automatically redeploy your site
